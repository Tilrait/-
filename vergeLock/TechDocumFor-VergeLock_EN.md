# VergeLock

**VergeLock** is an RTS project with elements of role-playing progression, crafting, and competitive modes, designed for session-based PvP battles between teams of players.
The main focus is not on mass unit production, but on individual control of a superhero and a limited, yet specialized squad that adapts to the owner's playstyle.

The project combines the tactical depth of RTS, the personalization of RPG, and the competitive dynamics of matchmaking, creating a balanced gameplay loop where victory depends not on quantity, but on the quality of decisions.

---

## Project Goal

To create a **multiplayer real-time strategy** with **RPG** elements, where each player controls a **unique superhero** and a **limited squad of units**, and character development, equipment crafting, and team tactics directly influence the match outcome.

---

## Project Description

- Genre: RTS / RPG / Multiplayer
- Control Type: direct hero control + unit micro-management
- Network Mode: PvP (Ranked / Casual / Custom)

---

- Automatic profile creation upon login via platforms or launchers.
- Unique player identifier (UID).
- Cloud synchronization of progress and statistics.

### 3. Superheroes
- Each hero has:
  - 3 active abilities;
  - 1 ultimate ability;
  - 1 passive ability.
- Roles: Tank / Damage / Support / Control / Hybrid.
- Customization and equipment modification capabilities.

### 4. Squads
- Unit types: melee, engineers, support.
- Commands: attack, defend, follow.

### 5. Crafting
- Creating items from resources gathered during a match.
- Item categories: weapons, armor.

### 6. Economy
- In-game store with support for various payment systems.

### 7. Interface
- Main Menu: Find Game / Heroes / Crafting / Store / Profile / Settings.
- HUD: health, abilities, minimap, score, timer.

### 8. Security
- Anti-cheat with client integrity checks.
- Trust Factor and reporting system.
- Account blocking for violations.
